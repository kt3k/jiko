# jiko

2024年（令和6年）交通事故統計データ

## データ

| ファイル             | 内容              |
| -------------------- | ----------------- |
| `honhyo_2024.csv`    | 本票（1〜12月）   |
| `hojuhyo_2024.csv`   | 補充票（1〜12月） |
| `kosokuhyo_2024.csv` | 高速票（1〜12月） |

## 出典

出典：警察庁ウェブサイト（https://www.npa.go.jp/publications/statistics/koutsuu/opendata/2024/opendata_2024.html）

本データは、警察庁が公開する交通事故統計オープンデータを加工して作成したものです（Shift_JIS
から UTF-8
へ変換）。利用条件は[公共データ利用規約 第1.0版](https://www.digital.go.jp/resources/open_data/pdl1.0)に準拠します。
